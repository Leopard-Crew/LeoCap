#LeoCap

Native Capture Utility for Mac OS X 10.5.8 Leopard / PowerPC

A Leopard-native screen capture, annotation and sharing utility.
Inspired by the workflow idea of ShareX, but implemented with
Cocoa, Quartz, Launch Services, Services and Automator integration.

LeoCap is not a C# or ShareX port.

ShareX may be used as a functional reference only.
All runtime, UI, capture and system integration code must be native
to Mac OS X 10.5.8 Leopard / PowerPC.
